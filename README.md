# Liaoluan (潦草清单) - 新野兽派极简效率工具

> **"拒绝精致的伪装，直面生活的粗粝。"**

**Liaoluan (潦草清单)** 是一款采用 **Neo-Brutalism (新野兽派)** 设计风格的 Android 个人效率应用。它摒弃了传统 Material Design 的圆润与柔和，用大胆的黑白高对比度、粗犷的边框和非传统的布局，为你提供一个最纯粹的任务、习惯与灵感管理平台。

## ✨ 核心功能详解

### 1. 任务清单 (Tasks) —— 每日待办的终结者
不仅仅是列出清单，更是对执行力的强制唤醒。

*   **⚡ 快速创建与编辑**
    *   极速录入任务名称与详细描述。
    *   支持对已创建任务进行二次编辑，随时调整目标。
*   **🔥 优先级视觉系统**
    *   **! 低优先级**：日常杂事，颜色温和 (BrutalColors.PriorityLow)。
    *   **!! 中优先级**：重要不紧急，醒目提示 (BrutalColors.PriorityMedium)。
    *   **!!! 高优先级**：紧急事项，强烈视觉冲击 (BrutalColors.PriorityHigh)，逼迫你立即行动。
*   **🏷️ 场景化标签**
    *   内置 **工作**、**生活**、**紧急**、**学习** 四大核心场景标签。
    *   首页可通过标签快速筛选，专注当下领域。
*   **⏰ 智能提醒与倒计时**
    *   **精确截止时间**：支持设定精确到秒的截止时间点。
    *   **动态倒计时**：任务卡片实时显示剩余时间（如 "剩 2小时"），营造紧迫感。
    *   **自定义提醒语**：拒绝机械的 "您有一个任务待完成"，你可以输入 **"再不写完就要由于了！"** 等个性化文字，在通知栏狠狠鞭策自己。
*   **✅ 归档与历史**
    *   左滑任务即可完成，完成后的任务会自动归档到底部或历史记录中，保持列表清爽。

### 2. 习惯养成 (Habits) —— 长期主义的见证者
用量化的方式，对抗人类的惰性。

*   **📅 灵活的排程系统**
    *   支持设置 **执行频率**：仅在周一、三、五提醒？没问题。
    *   **时间窗口**：设定习惯的 **开始时间** 和 **结束时间**（如 09:00 - 18:00），仅在有效时间内进行提醒。
*   **🔢 量化目标**
    *   支持设定 **每日目标值**（如 喝水 2000ml）和 **单位**（ml）。
    *   支持设定 **单次步进值**（如 每次喝 250ml），点击一次卡片即增加对应进度。
*   **🔔 强力提醒机制**
    *   **间隔提醒**：在时间窗口内，支持每隔 X 分钟（如 60分钟）循环提醒一次，直到当日目标完成。
    *   **纯净时间选择器**：专为习惯优化的时间选择器，剔除了繁琐的日期选择，只关注 "几点几分"。
*   **🔥 连胜 (Streak) 统计**
    *   直观展示当前连续坚持天数，断签一天，数据清零（部分逻辑），用损失厌恶心理激励坚持。
*   **💖 个性化动力**
    *   设置专属 **"自定义提醒语"**（如 "坚持运动，为了马甲线！"），让每一次通知都深具温度。
*   **📅 习惯日历与历史回溯 (New in v1.0)**
    *   **周视图日历**：在习惯页顶部集成了周日历组件，支持左右滑动切换日期。
    *   **历史回溯**：点击过去某一天的日期，即可查看当天的习惯完成情况（只读）。
    *   **未来规划**：查看未来日期的习惯安排，提前规划时间。
    *   **数据持久化**：所有的历史打卡记录均被完整保存，哪怕卸载重装（需配合备份功能）也不会丢失你努力过的凭证。

### 3. 灵感笔记 (Notes) —— 碎片思想的捕获器
大脑的外部硬盘，随时随地记录闪念。

*   **📝 极简编辑器**
    *   无干扰的纯文本编辑环境，专注于文字本身。
*   **🖼️图文混排**
    *   支持从系统相册 **插入图片**，自动压缩并嵌入笔记内容。
    *   笔记列表页支持 **智能预览**：仅展示前 3 行文字或首张图片，保持界面整洁。
    *   **一键展开/折叠**：点击展开按钮查看全文，无需频繁跳转页面。
*   **📌 重要置顶**
    *   支持将关键笔记 **置顶** 显示，永远停留在列表最上方。
*   **🎨 情绪标记**
    *   虽然是野兽派，但也关注心情。每条笔记可标记当时的心境。

### 4. 数据安全与迁移
*   **💾 本地备份与恢复**
    *   支持将所有数据（任务、习惯、习惯历史、笔记）导出为 JSON 文件。
    *   支持从 JSON 文件全量恢复数据，换机无忧。

## 🎨 设计哲学：Neo-Brutalism (新野兽派)

Liaoluan 的 UI 设计严格遵循新野兽派的美学原则：

1.  **高对比度 (High Contrast)**：以纯黑 (#000000) 和纯白 (#FFFFFF) 为主基调，辅以高饱和度的 蓝色 (HabitBlue)、黄色 (NoteYellow) 和 红色 (AlarmRed)。
2.  **粗轮廓 (Bold Borders)**：所有 UI 元素（卡片、按钮、输入框）均带有 2dp-4dp 的纯黑边框，拒绝模糊的阴影。
3.  **系统字体 (Default & Custom)**：集成了瘦身版的 **LXGW WenKai (霞鹜文楷)** 字体，在粗犷的框架中注入一丝人文气息。
4.  **直角与硬阴影**：卡片采用硬阴影（Offset Shadow），即黑色的实体色块作为阴影，营造出立体剪纸感。

## 🛠️ 技术架构

本项目是一个标准的现代 Android 工程，采用 **100% Kotlin** 编写。

*   **UI 框架**: Jetpack Compose (Material3)
*   **依赖注入**: Dagger Hilt
*   **架构模式**: MVVM (Model-View-ViewModel) + Clean Architecture
*   **本地存储**: Room Database (SQLite)
*   **后台任务**: Android WorkManager (用于处理复杂的定时提醒逻辑)
*   **异步编程**: Kotlin Coroutines & Flow
*   **图片加载**: Coil
*   **序列化**: Kotlinx Serialization

## 🏗️ 如何构建

1.  确保拥有 **Android Studio Koala** 或更高版本。
2.  克隆本项目：`git clone https://github.com/luzzr-123/liaoluan.git`
3.  等待 Gradle Sync 完成。
4.  连接 Android 调试设备或启动模拟器。
5.  点击 **Run 'app'**。
